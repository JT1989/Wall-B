PSEUDO CODE RELATED TO THE PROCEDURE OF THE CLIENT SERVER REQUEST LIFECYCLE

- Take input from users from the web browser

- store input so that it is retrievable, aka, persistence

- GET, POST, PUT, DELETE requests happening, application has to decide what happens based on any given type of request, 'routing', the process of receiving an external web request and then the application must make a decision about what code to execute based on the external request

- process the information based on preexisting logic/code

- send information back to the browser and route the client to correct destination.
